Модель (Model)
==============

`Модель (model)` - это экземпляр класса [CModel] или производного от него.
Модель используется для хранения данных и относящихся к ним бизнес-правил.

Модель представляет собой отдельный объект данных. Это может быть запись (row) таблицы базы данных или форма пользовательского ввода.
Каждое поле объекта данных представляется свойством (attribute) модели. Свойство (attribute) имеет метку (label) и может быть 
проверено (validated), используя набор правил.

Yii предоставляет 2 типа моделей: модель формы (form model) и Active Record. Оба типа являются расширением базового класса [CModel].

Модель формы (form model) - это экземпляр класса [CFormModel]. Модель формы используется для хранения данных, введенных пользователем.
Такие данные обычно бывают получены, использованы, а затем отброшены. Например, на странице авторизации модель такого типа может быть
использована для представления информации об имени пользователя и пароле. Для детального описания обратитесь к главе [Работа с формой (Working with Form)](/doc/guide/form.model)

Active Record (AR) - это шаблон проектирования, используемый для абстрагирования доступа к базе данных в объектно-ориентированной форме.
Каждый объект AR является экземпляром класса [CActiveRecord] или производного от него, представляя собой отдельую запись (row) в таблице
базы данных. Поля записи представляются свойствами объекта AR. Более детальная информация содержится в главе [Active Record](/doc/guide/database.ar).
