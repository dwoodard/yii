Что такое Yii
=============

Yii - это высокоэффективный, основанный на компонентной структуре PHP-фреймворк для разработки
масштабных веб-приложений. Он дает возможность масимально применять концепции повторного использования кода в
веб-программировании и может существенно ускорить процесс разработки. Имя Yii (произносится как `/i:/`) 
означает *простой (easy)*, *эффективный (efficient)* and *расширяемый (extensible)*.

Требования
----------

Для запуска веб-приложений на Yii вам понадобится веб-сервер с поддержкой PHP версии 5.1.0 или выше.

Для разработчиков, желающих использовать Yii, крайне полезным будет понимание концепций объектно-ориентированного программирования (OOP),
так как Yii  - это строго объектно-ориентированный фреймворк.

Для чего Yii будет лучшим выбором? 
----------------------------------

Yii - это фреймворк общего назначения для веб-программирования, который может быть использован для разработки
практически любых типов веб-приложений. Из-за своей легковесности и оснащенности продвинутыми средствами кэширования 
он в особенности подходит для разработки приложений, потребляющих большие объемы траффика, таких как порталы, форумы,
системы управления контентом (CMS), системы электронной коммерции и др.

Yii в сравнении с другими фреймворками
--------------------------------------

Подобно большинству других PHP-фреймворков, Yii - это MVC-фреймворк.

Yii превосходит другие фреймворки по своей эффективности, оснащенности и документированности. Yii изначально спроектирован 
очень тщательно для того, чтобы соответствовать всем требованиям при разработке серъезных веб-приложений. 
Yii не является ни побочным продуктом какого-нибудь проекта, ни беспорядочной смесью отдельных решений. Он является результатом
большого опыта авторов в разработке и исследовании веб-приложений, а также отражением наиболее популярных веб-фреймворков и приложений.
