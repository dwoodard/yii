Входной (entry) скрипт
======================

Входной скрипт - это PHP-скрипт, выполняющий первоначальную обработку пользовательского запроса.
Это единственный PHP-скрипт, который доступен для исполнения конечному пользователю по прямому запросу.

В большинстве случаев входной скрипт приложения на Yii содержит следующий простой код:

~~~
[php]
// удалите следующую строку для режима продакшна
defined('YII_DEBUG') or define('YII_DEBUG',true);
// подключаем файл инициализации Yii
require_once('path/to/yii/framework/yii.php');
// создаем экземпляр приложения и запускаем его
$configFile='path/to/config/file.php';
Yii::createWebApplication($configFile)->run();
~~~

Сначала скрипт подключает файл начальной загрузки фреймворка `yii.php`. 
Затем он создает экземпляр веб-приложения с необходимыми параметрами и запускает его на исполнение.

Режим отладки (Debug Mode)
--------------------------

Приложение на Yii может выполняться в режиме отладки или продакшна - в зависимости от значения константы `YII_DEBUG`. 

По умолчанию ее значение установлено в `false`, что означает режим продакшна. Для запуска в режиме отладки установите 
ее значение в `true` перед тем, как подключать файл `yii.php`.
Выполнение приложения в режиме отладки менее рационально, так как в этом случае приложение ведет множество внутренних логов.
С другой стороны, режим отладки очень полезен на стадии разработки, предоставляя большее количество отладочной
информации при возникновении ошибок.
